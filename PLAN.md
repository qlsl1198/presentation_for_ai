# AI 스타트업 프로젝트 기획서

## 1. 프로젝트명

**Presentation For AI**

> AI를 활용해 발표 주제 입력만으로 발표자료의 구조, 핵심 내용, 발표 스크립트를 자동 생성해주는 Streamlit 기반 발표 준비 서비스

---

## 2. 프로젝트 개요

현대 대학생과 직장인은 과제 발표, 세미나, 프로젝트 발표, 공모전 발표 등 다양한 상황에서 발표자료를 제작해야 한다. 그러나 발표자료 제작 과정은 많은 시간이 소요되며, 특히 주제 정리, 목차 구성, 핵심 문장 작성, 발표 흐름 설계에서 어려움을 겪는 경우가 많다.

**Presentation For AI**는 사용자가 발표 주제 또는 간단한 설명을 입력하면 AI가 발표자료의 목차, 슬라이드별 핵심 내용, 발표 스크립트를 자동으로 생성해주는 서비스이다.

본 프로젝트는 Python과 Streamlit을 활용하여 사용자가 브라우저에서 간단히 발표 주제를 입력하고, AI가 생성한 발표자료 초안을 바로 확인할 수 있도록 구현한다.

---

## 3. 기획 배경 및 문제 인식

발표자료를 만드는 과정에서 사용자는 다음과 같은 문제를 겪는다.

1. 발표 주제를 어떻게 구조화해야 할지 어렵다.
2. 슬라이드별 핵심 내용을 정리하는 데 시간이 오래 걸린다.
3. 발표 흐름이 자연스럽지 않아 전달력이 떨어진다.
4. 발표 대본을 따로 작성해야 해서 준비 시간이 증가한다.
5. 디자인보다 내용 구성 단계에서 막히는 경우가 많다.

특히 대학 과제나 팀 프로젝트에서는 발표자료를 짧은 시간 안에 제작해야 하는 경우가 많다. 따라서 AI를 활용해 발표 준비의 초기 과정을 자동화한다면 사용자의 시간 부담을 크게 줄일 수 있다.

---

## 4. 서비스 목표

본 프로젝트의 목표는 다음과 같다.

- 사용자가 입력한 주제를 기반으로 발표자료 구조를 자동 생성한다.
- 슬라이드별 제목과 핵심 내용을 AI가 제안한다.
- 발표자가 참고할 수 있는 발표 스크립트를 제공한다.
- 발표 준비 시간을 단축하고 발표 완성도를 높인다.
- Streamlit을 활용하여 간단하고 직관적인 AI 발표자료 생성 서비스를 구현한다.

- 발표 흐름이나 대본 작성이 어려운 학생
- 발표자료 초안을 빠르게 만들고 싶은 학생

### 5.2 직장인 및 예비 창업자

- 아이디어 제안서나 사업 발표자료를 구성해야 하는 사용자
- 발표 대본까지 함께 준비하고 싶은 사용자

### 5.3 교육자 및 강사

- 강의 자료의 초안을 빠르게 구성하고 싶은 사용자

## 6. 핵심 기능

### 6.1 발표 주제 입력 기능

사용자는 발표하고 싶은 주제나 간단한 설명을 입력한다.

예시 입력:

```text
AI를 활용한 대학생 발표자료 자동 생성 서비스
```

AI는 입력된 주제를 분석하여 발표자료에 적합한 구조를 생성한다.

---

### 6.2 발표 조건 입력 기능

사용자는 발표자료 생성에 필요한 추가 조건을 입력할 수 있다.

예시 입력 항목:

```text
발표 대상: 대학생
발표 시간: 5분
발표 목적: 과제 발표
발표 분위기: 이해하기 쉽고 실용적인 발표
```

이 정보를 활용하여 AI가 사용자의 상황에 맞는 발표자료 초안을 생성한다.

---

### 6.3 AI 목차 생성 기능

입력된 주제를 기반으로 발표자료의 전체 목차를 자동 생성한다.

예시 결과:

```text
1. 서비스 소개
2. 문제 인식
3. 핵심 기능
4. 기대 효과
5. 시장성
6. 향후 발전 방향
```

이 기능은 발표 흐름을 잡기 어려운 사용자를 돕는 핵심 기능이다.

---

### 6.4 슬라이드별 내용 생성 기능

AI는 각 목차에 맞는 슬라이드 제목과 핵심 내용을 생성한다.

예시:

```text
슬라이드 제목: 문제 인식

핵심 내용:
- 발표자료 제작에는 많은 시간이 소요됨
- 발표 흐름 구성과 핵심 문장 작성이 어려움
- AI를 활용하면 초안 작성 시간을 줄일 수 있음
```

이를 통해 사용자는 빈 화면에서 시작하지 않고, AI가 만든 초안을 바탕으로 발표자료를 수정하거나 보완할 수 있다.

---

### 6.5 발표 스크립트 생성 기능

각 슬라이드에 맞는 발표 대본을 자동으로 생성한다.

예시:

```text
이 슬라이드에서는 발표자료 제작 과정에서 사용자가 겪는 문제를 설명하겠습니다.
많은 학생과 직장인은 발표자료를 만들 때 목차 구성과 핵심 내용 정리에 어려움을 겪습니다.
따라서 본 서비스는 AI를 활용해 이러한 과정을 자동화하고자 합니다.
```

발표 스크립트는 사용자가 실제 발표를 준비할 때 바로 참고할 수 있다.

---

### 6.6 결과 출력 기능

생성된 결과는 Streamlit 화면에 다음과 같은 형태로 제공된다.

- 발표 제목
- 전체 목차
- 슬라이드별 핵심 내용
- 슬라이드별 발표 스크립트
- 복사 가능한 텍스트 결과

사용자는 생성된 결과를 확인한 뒤 필요한 부분을 수정하여 발표자료 제작에 활용할 수 있다.

## 7. AI 활용 방식

본 서비스는 생성형 AI를 활용하여 사용자의 입력을 분석하고 발표자료 초안을 생성한다.

### 7.1 입력 데이터

사용자가 직접 입력한 발표 주제, 발표 대상, 발표 시간, 발표 목적 등의 데이터를 사용한다.

예시:

```text
주제: AI를 활용한 발표자료 자동 생성 서비스
대상: 대학생
발표 시간: 5분
목적: 서비스 기획 발표
```

### 7.2 AI 처리 과정

1. 사용자의 입력 주제 분석
2. 발표 대상과 발표 목적 파악
3. 발표 시간에 맞는 목차 구성
4. 각 목차에 맞는 슬라이드 내용 생성
5. 발표 흐름에 맞는 발표 스크립트 생성
6. 생성 결과를 Streamlit 화면에 출력

### 7.3 AI 기능의 의미

본 프로젝트에서 AI는 단순히 문장을 생성하는 도구가 아니라, 발표자료 제작 과정에서 가장 중요한 **구조화**, **내용 요약**, **발표 흐름 설계**, **발표 대본 작성**을 지원하는 핵심 기능으로 활용된다.

## 10. 화면 구성

### 10.1 메인 화면

메인 화면은 사용자가 발표자료 생성을 위해 필요한 정보를 입력하는 화면이다.

구성 요소:

- 서비스 이름
- 서비스 소개 문구
- 발표 주제 입력창
- 발표 대상 입력창
- 발표 시간 입력창
- 발표 목적 입력창
- 발표자료 생성 버튼

### 10.2 결과 화면

결과 화면은 AI가 생성한 발표자료 초안을 보여주는 화면이다.

구성 요소:

- 생성된 발표 제목
- 전체 발표 목차
- 슬라이드별 제목
- 슬라이드별 핵심 내용
- 슬라이드별 발표 스크립트
- 생성 결과 확인 영역

Streamlit의 입력 위젯과 출력 컴포넌트를 활용하여 사용자가 쉽게 발표자료 초안을 생성하고 확인할 수 있도록 구성한다.

## 14. 한계점 및 개선 방향

### 14.1 한계점

- AI가 생성한 내용이 항상 정확하지 않을 수 있다.
- 사용자의 입력이 부족하면 결과 품질이 낮아질 수 있다.
- 디자인이 포함된 완성형 PPT 생성까지는 추가 개발이 필요하다.
- 생성된 발표자료는 사용자의 검토와 수정이 필요하다.
- AI API 사용량에 따라 비용이 발생할 수 있다.

### 14.2 개선 방향

- PPT 파일 자동 생성 기능 추가
- 발표자료 디자인 템플릿 적용 기능 추가
- 사용자가 업로드한 문서 기반 발표자료 생성 기능 추가
- 발표 시간에 맞춘 스크립트 길이 조절 기능 추가
- 생성 결과 저장 및 다시 불러오기 기능 추가
- 다양한 발표 유형별 프롬프트 제공
- 생성된 발표자료 다운로드 기능 추가
